### 🎯 A股数据获取速查表





| 功能类别               | 函数名称                                            | 主要参数                                                     | 返回数据类型                                                 | 核心作用                                     |
| :--------------------- | :-------------------------------------------------- | :----------------------------------------------------------- | :----------------------------------------------------------- | :------------------------------------------- |
| **获取股票列表**       | `ak.stock_info_a_code_name()`                       | 无                                                           | DataFrame，包含 `code` (股票代码) 和 `name` (股票名称)       | 获取所有A股的基本信息，是批量获取数据的起点  |
| **获取实时行情**       | `ak.stock_zh_a_spot()` 或 `ak.stock_zh_a_spot_em()` | 无                                                           | DataFrame，包含 `代码`、`名称`、`最新价`、`涨跌幅` 等实时交易信息 | 获取全市场或指定股票的实时快照               |
| **获取日线历史行情**   | `ak.stock_zh_a_hist()`                              | `symbol` (股票代码), `period` (周期), `start_date`, `end_date`, `adjust` (复权方式) | DataFrame，包含 `日期`、`开盘`、`收盘`、`最高`、`最低`、`成交量` 等 | 获取个股详细的历史K线数据，是量化分析的基础  |
| **获取分钟级历史数据** | `ak.stock_zh_a_hist_min_em()`                       | `symbol`, `period` (如 `1`, `5`, `15`, `30`, `60` 分钟), `start_date`, `end_date` | DataFrame，包含 `时间`、`开盘`、`收盘`、`最高`、`最低`、`成交量`、`成交额` 等 | 获取个股更精细的分钟级历史数据，用于短线分析 |
| **获取分时数据**       | `ak.stock_zh_a_tick_tx()`                           | `symbol`, `trade_date` (交易日期)                            | DataFrame，包含每一笔交易的详细数据，如 `时间`、`价格`、`成交量`、`成交额` 等 | 获取指定日期最精细的逐笔成交数据             |
